O projeto "WeatherApp" é uma aplicação web simples que permite aos usuários verificar o clima de uma cidade digitando o nome da cidade desejada. Utilizando a API do OpenWeatherMap, a aplicação exibe informações meteorológicas atualizadas, como temperatura, descrição do clima, umidade, velocidade do vento, e exibe também um ícone representativo do clima atual. Além disso, a aplicação utiliza a API Flags para exibir a bandeira do país correspondente à cidade pesquisada. O projeto é construído com JavaScript puro para manipulação do DOM e consultas à API, garantindo uma experiência de usuário rápida e eficiente.
